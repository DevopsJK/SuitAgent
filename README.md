## 容器监控

在这个Docker盛行的容器时代，监控自然也要跟上步伐。不仅应用运行在容器里，要是连采集各种监控指标的Agent也能够运行在容器里，而且还能同时具备宿主机的监控以及其他容器内应用的自动监控，那岂不爽哉。恭喜你，SuitAgent已上线容器运行环境监控采集。具体详情移步[这里](https://github.com/DevopsJK/SuitAgent/wiki/containerRuntime)
